# 台北漫畫博覽會一日行程

可離線使用的單頁行程網頁，內容涵蓋 2026/07/25 新竹—台北漫畫博覽會—Banco Pizza—（可選）國立歷史博物館—南港晚餐—高鐵南港站回程的行程。

## 開啟方式

直接雙擊 `index.html` 即可使用所有行程、導航連結、勾選與備忘功能。行程勾選和備忘會儲存在目前瀏覽器的本機資料中。

Android 手機請使用 `taipei-comic-expo-android-offline.html`。它將 CSS、JavaScript 與兩張圖片都嵌在同一個檔案內，不會受到 Android HTML 檢視器禁止相對路徑資源的影響。部分檢視器可能禁止本機儲存；此時行程仍可使用，但勾選與備忘不會在關閉後保留。

## 離線與安裝

網頁內容本身不依賴外部套件。若要啟用瀏覽器的「安裝」與 Service Worker 離線快取功能，請用本機靜態伺服器開啟專案，例如在此資料夾執行：

```powershell
py -m http.server 8080
```

再開啟 `http://localhost:8080`。Google Maps、官方資訊與天氣按鈕需要網路連線。

## 已核對的關鍵資訊（2026/07/23）

- 漫畫博覽會：2026/07/23–07/27，世貿一館。
- 國立歷史博物館《埃及木乃伊—永生傳說》：2026/06/18–09/28，1、2 樓展廳；開放時間 10:00–18:00。
- 台北101／世貿站至台北車站：捷運票價 NT$25、行車約 10 分鐘；世貿一館旁為 R03 5 號出口。
- 世貿或國立歷史博物館至南港站：捷運票價 NT$30；世貿出發約 30 分鐘，史博館出發含步行約 45–50 分鐘。

活動入場規則、商品販售、列車與捷運時刻可能調整，當日以官方公告與現場指示為準。
