# 🧳 Travel Itinerary Archive | 個人出遊行程紀錄與規劃庫

這個儲存庫（Repository）用於歸檔與託管歷次出遊的行程規劃網頁。透過 AI 輔助生成結構化的 HTML 頁面，並利用 GitHub Pages 進行自動化部署，打造個人專屬的動態旅遊日誌。

---

## 📌 專案簡介

* **主要功能**：集中管理所有歷史與未來的旅遊行程，提供結構化的景點、交通與時間規劃。
* **部署方式**：透過 GitHub Pages 發布為靜態網站，便於在手機上隨時查閱或與同行夥伴分享。
* **架構特色**：採用多頁面（Multi-page）目錄結構，保留所有歷史行程，不因新行程而覆蓋舊紀錄。

---

## 📂 檔案結構

```text
.
├── index.html              # 旅遊總覽入口頁（連結至各個行程）
├── assets/                 # 全站通用靜態資源（CSS、共用圖片等）
├── trips/                  # 所有行程的歸檔目錄
│   ├── 2026-04-somewhere/      # 單次行程資料夾（範例：2026 某處）
│   │   ├── index.html      # 該次行程的主頁面
│   │   └── assets/         # 該次行程專用的圖片或資源
│   └── 2026-10-elsewhere/      # 單次行程資料夾（範例：2026 某處）
│       └── index.html
└── README.md               # 專案說明文件
