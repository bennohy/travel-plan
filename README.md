# 🧳 Personal Travel Archive | 個人旅行日誌與行程庫

這個儲存庫（Repository）用於歸檔與展示個人歷次出遊的行程規劃網頁。透過 AI 輔助生成網頁內容與 JSON 資料庫，並利用 GitHub Pages 進行自動化託管，打造個人專屬的動態旅遊首頁。

---

點擊下方連結即可直接瀏覽行程總覽首頁：

👉 **[開啟我的旅行總覽頁](https://bennohy.github.io/travel-plan/)**

---

## 📌 專案特色

* **暖色手帳風格**：總覽頁採用奶油白與大地色系設計，提供舒適溫暖的瀏覽體驗。
* **JSON 動態渲染**：總覽頁（`index.html`）自動讀取 `trips.json`，新增行程時無需修改任何 HTML 程式碼。
* **靈活相容性**：支援圖片封面（`cover`），若無圖片則自動優雅降級為質感純文字卡片。

```text

├── index.html              # 旅遊總覽入口頁（連結至各個行程）

│       └── index.html
